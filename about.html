About

Why I made this page?

This webpage was assigned as a project milestone for my Intro to Web development class. This is a link to the class webpage.

After watching a plethora of youtube videos, course slides, and much googling, I have finally grasped the base level skills needed to create this webpage, however, there is much more I need to learn.

When it comes to scripting the HTML for this website, the simple html format of title, head, body, and paragraphs were pretty easy to understand. It was implementing the CSS and JavaScript that threw me for a bit of a loop.

Style Inspiration

The CSS style and inspiration of this webpage comes from a mix of a stylesheet template that I saw used on w3schools.com and a kind of techno/cyberpunk theme.

As time goes on, I hope to upgrade this page with more advanced CSS and Javascript to fit that techno theme, including animations and more interactive features.
