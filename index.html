My First Webpage: History of the Web

Welcome

This is my very first website, built as part of a class project for IT 3203: Intro to Web Development at Kennesaw State University. This is the class website. I'm proud to showcase the skills I've learned including HTML, CSS, and JavaScript.

To explore the site, click on the three-line menu in the top right corner. From there, you can visit different sections of the site to learn about the history and evolution of the World Wide Web.

Summary

This site walks you through the stages of how the Web came to be:

  • About: Origins of the Internet and early protocols
  • Web 1.0: Simple static pages and browsers like Mosaic
  • Web 2.0: Interactive platforms, social media, mobile evolution
  • Web 3.0: Blockchain, AI, virtual reality, and the future of the web
  • Resources: The sources and tools used for this project
